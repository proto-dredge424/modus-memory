## ✍️ Working With Markdown

modus-memory uses plain markdown, which is simple text with light formatting.

You can use:

- `#` for a title
- `-` for a list
- `**bold**` for strong text
- Line breaks for new sections

Example:

# Project Memory

- User likes short replies
- Save meeting notes here
- Keep task history in this file

This format stays easy to read and easy to edit.
